# 🎹 快捷键大全

> **完整快捷键参考手册**
>
> `<leader>` = 空格键（Space）。例如 `<leader>ff` = 先按 Space，再按两次 f。

| 快捷键 | 功能 |
|--------|------|
| `<leader>o` / `-` | 打开 oil / 回上级目录 |

*仅 `.csv` / `.tsv` 文件内生效。*

| 命令 | 作用 |
|------|------|

| `vif` / `vaf` | 选中字段 |
