## Quick start

1. `nvim .` — opens dashboard
2. `<leader>ff` — find a file
3. `gd` — jump to definition
4. `<Tab>` — switch files
5. `<leader>th` — change theme if you want

Everything auto-installs on first run (LSP servers, parsers, formatters). Just wait a few seconds.
