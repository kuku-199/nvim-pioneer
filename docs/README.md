## 🤝 Contributing to Documentation

Want to improve our docs? See our [Contributing Guide](../CONTRIBUTING.md) for guidelines.

**Quick start for docs:**
1. Fork the repository
2. Create a branch: `git checkout -b docs/update-keybindings`
3. Edit the Markdown file
4. Commit with clear message: `docs: improve keybindings guide`
5. Submit a Pull Request
