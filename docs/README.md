## Quick start

1. `nvim .` — opens sci-fi HUD dashboard
2. `<leader>ff` — find a file (Telescope)
3. `s` + label — jump anywhere (Flash)
4. `gd` — go to definition (LSP)
5. `<leader>xx` — see diagnostics (Trouble)
6. `<Tab>` — switch buffers
7. `<leader>th` — cycle themes

Everything auto-installs on first run (LSP servers, parsers, formatters).
