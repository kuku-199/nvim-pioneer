# ⌨️ Neovim 快捷键速查表

> 位置：`~/.config/nvim/CHEATSHEET.md`
>
> 打开本文件：在 nvim 里按 `<leader>ff` 然后输 `CHEATSHEET`

---

## 1. 基本操作

| 快捷键 | 模式 | 功能 |
|--------|:----:|------|
| `jk` / `kj` | 插入 | 退出插入模式（同 Esc） |
| `jj` | 插入 | 同上 |
| `<C-s>` | 普通 | **保存**文件 |
| `<C-q>` | 普通 | 退出 |
| `<C-z>` | — | 撤销（vim 默认） |
| `u` | 普通 | 撤销 |
| `<C-r>` | 普通 | 重做 |

## 3. 📂 文件浏览

### 3.1 neo-tree（树状文件浏览器）

| 快捷键 | 功能 |
|--------|------|
| `<leader>e` | **打开/关闭** 文件树 |
| `<leader>E` | 在文件树里**定位**当前文件 |
| `q` | （文件树内）关闭面板 |
| `h` | 折叠目录 |
| `l` 或 `Enter` | 打开文件或展开目录 |
| `v` | **垂直**分屏打开 |
| `s` | **水平**分屏打开 |
| `t` | 在新标签页打开 |
| `H` | 显示/隐藏隐藏文件 |
| `R` | 刷新 |
| `P` | 预览文件 |
| `/` | 在文件树里模糊搜索 |
| `<Space>` | 展开/折叠节点 |
| `?` | 显示帮助 |

### 3.2 oil（目录编辑）

| 快捷键 | 功能 |
|--------|------|
| `<leader>o` | **打开** oil |
| `-` | 回到上级目录 |
| `Enter` | 打开文件/进入目录 |
| `<C-v>` | 垂直分屏打开 |
| `<C-s>` | 水平分屏打开 |
| `g.` | 显示/隐藏隐藏文件 |
| `gs` | 改变排序方式 |
| `gx` | 用系统应用打开 |
| `~` | 跳转到 home 目录 |
| `` ` `` | 跳转到工作目录 |
| `<C-l>` | 刷新 |

**Oil 妙用**：在 oil 里可以直接 `dd` 删除文件、`yy` 复制、`p` 粘贴、`R` 重命名——像编辑文本一样操作文件。

## 7. 🌲 Treesitter 语法操作

### 7.1 文本对象（选择代码结构）

| 快捷键 | 功能 |
|--------|------|
| `yaf` | 复制**整个函数**（yank a function） |
| `daf` | 删除整个函数 |
| `cif` | 修改函数内部内容 |
| `yac` | 复制整个类 |
| `yif` | 复制函数内部（不包含函数声明） |
| `cia` | 修改函数参数 |
| `yia` | 复制参数 |
| `yab` | 复制整个代码块 |

> 用 `d`（删除）、`c`（修改）、`y`（复制）+ `a`（包括边界）/ `i`（内部）+ `f`（函数）/ `c`（类）/ `a`（参数）/ `b`（块）

### 7.2 跳转

| 快捷键 | 功能 |
|--------|------|
| `]m` | 跳到**下一个**函数开头 |
| `[m` | 跳到**上一个**函数开头 |
| `]]` | 跳到下一个类开头 |
| `[[` | 跳到上一个类开头 |
| `]f` | 跳到下一个函数调用 |
| `[f` | 跳到上一个函数调用 |

### 7.3 参数交换

| 快捷键 | 功能 |
|--------|------|
| `<leader>a` | 把当前参数和后一个**交换** |
| `<leader>A` | 把当前参数和前一个交换 |

### 7.4 增量选择

| 快捷键 | 功能 |
|--------|------|
| `<c-space>` | 从当前光标**逐级扩大**选中范围（第一次按开始，继续按扩大） |
| `<c-s>` | 扩大到当前作用域 |

**示例**：光标在函数名上 → 按 `<c-space>` → 选中函数名 → 再按 → 选中整个函数 → 再按 → 选中类 → ...

## 9. 🔖 Buffer（文件标签）管理

| 快捷键 | 功能 |
|--------|------|
| `<Tab>` | 切换到**下一个**文件 |
| `<S-Tab>` | 切换到**上一个**文件 |
| `<leader>x` | 关闭当前文件 |
| `<leader>b` | 创建**新**的空白文件 |
| `<leader>bC` | 关闭**其他**所有文件 |
| `<leader>bp` | 固定标签（不会被关闭） |
| `Alt+,` | 把当前标签左移 |
| `Alt+.` | 把当前标签右移 |

**标签栏（顶部）说明：**
```
▎ init.lua  ▎ cheatsheet  ▎ lsp.lua  ▎ 󰅖
────────────────────────────────────────────
```
- `▎` 粗竖线 = 当前激活的文件
- `󰅖` 图标 = 可关闭
- 文件过多时自动折叠，用 `<Tab>` 翻页

## 14. 📊 CSV/TSV 表格查看（csvview.nvim）

打开 `.csv` / `.tsv` 文件后，用以下命令：

| 命令 | 作用 |
|------|------|
| `:CsvViewToggle` | 开关表格视图 |
| `:CsvViewToggle display_mode=border` | 切换到竖线表格模式 |
| `:CsvViewToggle display_mode=highlight` | 切回高亮模式 |
| `:CsvViewToggle header_lnum=none` | 无表头 |
| `:CsvViewInfo` | 查看 CSV 统计（分隔符、行列数） |

**csvview 启用后，Excel 式导航：**

| 快捷键 | 功能 |
|--------|------|
| `<Tab>` | 跳到下一个字段末尾 |
| `<S-Tab>` | 跳到上一个字段末尾 |
| `<Enter>` | 跳到下一行同字段 |
| `<S-Enter>` | 跳到上一行同字段 |
| `vif` | 选中当前字段内容 |
| `vaf` | 选中整个字段（含分隔符） |
| `dif` | 删除字段内容 |

## 15. 🔄 配置结构（知道文件在哪）

```
~/.config/nvim/
├── init.lua                  ← 入口
├── CHEATSHEET.md             ← 本文件
├── lazy-lock.json            ← 插件版本锁定（自动生成）
├── lua/config/
│   ├── options.lua           ← 基础设置
│   ├── keymaps.lua           ← 全局快捷键
│   ├── lazy.lua              ← 插件管理器入口
│   ├── theme.lua             ← 主题切换逻辑
│   ├── shortcuts.lua         ← Ctrl+C/V/A 快捷键
│   └── plugins/
│       ├── coding/
│       │   ├── blink.lua     ← 补全引擎（blink.cmp）
│       │   ├── lsp.lua       ← LSP 配置
│       │   └── rust.lua      ← Rust 专用配置
│       ├── editor/
│       │   ├── autopairs.lua ← 自动括号
│       │   ├── comment.lua   ← 注释快捷键
│       │   ├── conform.lua   ← 保存自动格式化
│       │   ├── gitsigns.lua  ← Git 行内改动标记
│       │   ├── markdown.lua  ← Markdown 预览
│       │   ├── neo-tree.lua  ← 文件树
│       │   ├── oil.lua       ← 目录编辑
│       │   └── treesitter.lua← 语法树增强
│       ├── embedded/
│       │   ├── clangd.lua    ← C/C++ 增强
│       │   ├── cmake.lua     ← CMake 集成
│       │   └── dap.lua       ← 调试器
│       ├── tools/
│       │   ├── telescope.lua ← 模糊搜索
│       │   ├── csvview.lua   ← CSV/TSV 表格显示
│       │   ├── terminal.lua  ← 浮动终端
│       │   └── which-key.lua ← 快捷键提示
│       └── ui/
│           ├── bufferline.lua← 顶部标签栏
│           ├── dashboard.lua ← 启动页
│           ├── indent.lua    ← 缩进线
│           ├── lualine.lua   ← 状态栏（含 CPU/RAM 监控）
│           └── themes.lua    ← 主题包
```
